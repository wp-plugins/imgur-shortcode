== Description ==
Embed an imgur image in to any post or page with easy using shortcodes. All you need is the ID form the image. 


[Get Support](http://fitbobcat.com) 


== Installation ==


This plugin is easy to install like other plug-ins of Wordpress as you need to just follow the below mentioned steps:

1. upload imgur-shortcode folder to wp-Content/plugins folder.

2. Activate the plugin from Dashboard / Plugins window.


### Usage ###
1. Get url id from imgur photo. (if the link is http://imgur.com/FYGIRkb, then the id is FYGIRkb.gif or .jpg
2. Copy and paste "[imgur id="FYGIRkb.jpg" /] in to your blog post or page
3. Save!
